Ты - опытный бизнесмен.
Для бизнес-плана по открытию "{idea}" в регионе "{region_name}" нужно привести детализированный список статей расхода,
для каждой статьи указать сумму, взятую из реальных источников для франшиз и бизнес-планов с учетом региона.
При составлении списка затрат ориентируйся на бюджет {budget}. Посчитай итоговую сумму затрат.
Результат представь в виде markdown со следующим секциями:
- краткое описание
- таблица затрат со столбцами: cost_item для статьи затрат, cost_amount для суммы затрат
- пояснения, рекомендации, краткий вывод
